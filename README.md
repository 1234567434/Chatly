# 💬 Chatly Messenger

Современный мессенджер с премиум подпиской **Chatly PRO**, построенный на Node.js + Socket.io.

## 🚀 Запуск

```bash
cd chatly
npm install
npm start
```

Откройте **http://localhost:3000** в браузере.

## 🌍 Языки

Мессенджер поддерживает **2 языка** с мгновенным переключением:
- 🇷🇺 **Русский** (по умолчанию)
- 🇺🇦 **Украинский**

Переключатель языка находится на экране входа и в сайдбаре.

## 📱 Функционал

### Основной
- 🔐 Регистрация / Авторизация (JWT + bcrypt)
- 💬 Реалтайм сообщения (WebSocket Socket.io)
- 🔍 Поиск пользователей
- 😀 Эмодзи-пикер (5 категорий, 400+ эмодзи)
- 📱 Адаптивный дизайн (мобильный + десктоп)
- 💾 Сохранение сообщений в файл
- 🟢 Онлайн / Оффлайн статусы
- ⌨️ Индикатор «печатает...»
- 📋 Контекстное меню сообщлений

### 💎 Chatly PRO
- 🎨 **6 кастомных тем**: Default, Midnight, Ocean, Sunset, Forest, Neon
- 😍 **Реакции** на сообщения (двойной клик или контекстное меню)
- ✅ **Статус прочитания** (✓✓ синие галочки)
- 📌 **Закрепление** сообщлений в чате
- 💎 **PRO бейдж** рядом с именем
- 🏷️ **Приоритетный статус**

## 💎 Как получить PRO

1. Зайдите в **⚙️ Настройки** → вкладка **💎 PRO**
2. Нажмите **«Отправить запрос на PRO»**
3. В консоли сервера появится запрос
4. Введите в консоли:
   ```
   pro <username>
   ```
5. Пользователь мгновенно получит PRO!

## 🎮 Консольные команды сервера

| Команда | Описание |
|---------|----------|
| `pro <username>` | Выдать PRO подписку |
| `remove-pro <username>` | Забрать PRO |
| `list` | Список всех пользователей |
| `requests` | Запросы на PRO |
| `online` | Кто онлайн |
| `help` | Помощь |

## 🛠️ Технологии

- **Сервер**: Node.js, Express, Socket.io
- **Клиент**: Vanilla JS, CSS3 (анимации, glassmorphism)
- **Интернационализация**: RU / UA с мгновенным переключением
- **Auth**: JWT + bcrypt
- **Данные**: JSON файловое хранилище
